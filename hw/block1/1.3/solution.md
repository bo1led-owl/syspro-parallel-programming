# Диапазоны значений переменных

Утверждается, что для любой трассы:
- `z == 1`
- `0 <= x <= 1`
- `0 <= y <= 2`

Притом `y <= 1 => x == 1`.

Таким образом, возможные результаты:
- `(0, 2, 1)`
- `(1, 2, 1)`
- `(1, 0, 1)`
- `(1, 1, 1)`

Покажем, что это действительно все возможные результаты:

## `z == 1`
Т.к. `z` модифицируется ровно один раз (`B.4`), `z` не может содержать результатов кроме `1`

## `0 <= x`
Единственный вариант, когда `x` становится меньше нуля --- исполнение `C.4` при `x == 0`, т.е. во время исполнения `C.3` `x` был равен нулю. `C.3` и `C.4` исполняются только при `y == 2`.

Единственная запись в `y` --- `A.3`. 

Если `y == 2`, то `A.3` исполнилась после `B.4`, а значит и `B.2`.

Единственная запись в `x` --- `B.2`.

Если `x` в `C.3` оказался равен нулю, то `C.3` исполняется раньше, чем `B.2`.

Получаем зависимости:
- `y == 2 => ... -> B.2 -> ... -> A.3 -> ...`
- `x == 0 => ... -> C.3 -> ... -> B.2 -> ...`

Но при этом `C.3` достижима только при `y == 2`, т.е. `... -> B.2 -> ... -> B.4 -> ... -> A.3 -> ... -> C.2 -> ... -> C.3 -> ...`

Видно противоречие между `... -> C.3 -> ... -> B.2 -> ...` и `... -> B.2 -> ... -> C.3 -> ...`.

## `x <= 1`
Единственные изменения `x` --- `B.2`, дающая `+1`, и `C.4`, дающая `-1`. Каждая исполняется единожды, следовательно ничего больше `1` получить невозможно.

## `0 <= y <= 2`
Единственная запись в `y` --- `A.3`. Диапазон значений `x` и `z` нам известен. Учтем лишь, что возможно исполнение `A.1 -> A.2 -> A.3 -> ... -> B.4 -> ...`, при котором на момент `A.3` `z` равняется нулю.

## `y <= 1 => x == 1`
Положим, что в итоге `y <= 1`, но `x == 0`.

Раз `x == 0`, `C.4` произошла. Тогда на момент `C.2` `y == 2`. Т.к. `y` только возрастает, получаем противоречие с `y <= 1`.

# Трассы

## `(0, 2, 1)`
`B.1 -> B.2 -> B.3 -> B.4 -> A.1 -> A.2 -> A.3 -> C.1 -> C.2 -> C.3 -> C.4`

## `(1, 2, 1)`
`C.1 -> C.2 -> B.1 -> B.2 -> B.3 -> B.4 -> A.1 -> A.2 -> A.3`

## `(1, 0, 1)`
`A.1 -> A.2 -> A.3 -> B.1 -> B.2 -> B.3 -> B.4 -> C.1 -> C.2`

## `(1, 1, 1)`
`A.1 -> B.1 -> B.2 -> B.3 -> B.4 -> A.2 -> A.3 -> C.1 -> C.2`
